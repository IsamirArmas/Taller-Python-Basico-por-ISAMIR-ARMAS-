##  Descripción del Taller

Este repositorio contiene la información general y el plan de estudios del taller de introducción a Python. Este programa está diseñado para estudiantes y personas que desean iniciar en el mundo de la programación sin experiencia previa.

El enfoque principal es brindar una introducción práctica y directa, utilizando la consola interactiva (REPL) para resolver problemas fundamentales y desarrollar la lógica computacional paso a paso, sin depender de interfaces gráficas complejas en una etapa inicial.

##  Objetivos de Aprendizaje

Al finalizar las sesiones, los participantes tendrán la capacidad de:
* Comprender los fundamentos del lenguaje y su sintaxis.
* Configurar y utilizar el entorno de desarrollo básico en su computadora.
* Dominar la declaración de variables y operaciones de lógica matemática.
* Implementar lógica algorítmica mediante estructuras de control condicionales (if/else) y bucles (for, while).
* Modularizar el código creando y llamando funciones sencillas.
* Desarrollar proyectos interactivos manejando la entrada y salida de datos en consola.

##  Público Objetivo

El contenido está estructurado y dirigido a:
* Estudiantes universitarios de cualquier carrera técnica o científica.
* Profesionales e individuos curiosos sobre el desarrollo de software.
* Personas que buscan conocer herramientas modernas para automatizar tareas diarias.

## ⚙️ Requisitos Previos (Entorno de Trabajo)

Para participar en los ejercicios prácticos, es necesario contar con:
1. **Intérprete de Python:** Versión 3.x instalada en el sistema operativo.
2. **Editor de Código:** Visual Studio Code (recomendado) o cualquier IDE de preferencia.
3. **Consola/Terminal:** Acceso a la línea de comandos del sistema operativo (CMD, PowerShell o Terminal).

## 📚 Bibliografía y Recursos de Apoyo

Para expandir los conocimientos adquiridos, se recomienda consultar las siguientes fuentes:
* **Documentación Oficial:** El sitio web oficial de Python en español.
* **Plataformas de Práctica Interactivas:** Entornos en línea como Replit o Trinket para programar desde el navegador.
* **Literatura:** Libros introductorios de automatización y fundamentos básicos del lenguaje.
